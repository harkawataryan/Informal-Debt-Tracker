Informal Debt Tracker

It's REST. Use /api/* endpoints.

  • GET /api/people
  • POST /api/people — { name, email }
  • GET /api/debts?status=OPEN|SETTLED
  • POST /api/debts — { fromId, toId, amount, dueDate, note }
  • POST /api/debts/{id}/settle
  • POST /api/reminders/send — { daysAhead }

For local email, check MailHog UI.
